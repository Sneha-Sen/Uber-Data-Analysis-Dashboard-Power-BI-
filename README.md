# Uber Data Analysis Dashboard (Power BI)

## üìå Project Overview
This Power BI dashboard provides a comprehensive analysis of Uber's ride performance, focusing on the "Auto" segment. It tracks key performance indicators (KPIs) such as booking success rates, revenue trends, and customer satisfaction to help drive operational efficiency.

## üìä Key Insights & Features
* **Executive Summary:** High-level view of 93K completed bookings and $52M in total revenue.
* **Operational Health:** Tracked a 57K "Lost Bookings" count to identify potential supply-demand gaps.
* **Revenue Segmentation:** Breakdown of revenue by vehicle type (Auto, Bike, Go Mini, etc.), with 'Auto' leading at $13M.
* **Temporal Trends:** Monthly analysis of bookings and revenue to identify seasonal peaks (notably high performance in March and October).
* **Geographic Hotspots:** Identification of Khandsa as the top pickup point and Ashram as the top drop-off location.
* **Rating Analysis:** Monitoring of Average Customer Rating (4.40) vs. Driver Rating (4.23).
